CafeLocate Academic References

Purpose
This file contains article/paper-style references that are suitable for a student project report, dissertation, or bibliography section.
These are the references you should prefer when your teacher asks for "articles" or "papers."

How to use this file
1. Use these references in your project report/reference section.
2. Use ASSUMPTIONS_AND_REFERENCES.txt for implementation traceability.
3. Use IMPLEMENTATION_DOC_REFERENCES.txt for library/API documentation sources.

9. Honest Citation Note
================================================================================

These papers support the methods and design choices used in the project.
However, not every exact numeric weight in the implemented formulas came directly from a published paper.
Some final formulas in the code are project-specific combinations of standard ideas:
- weighted scoring
- distance-based accessibility
- geospatial suitability aggregation
- ensemble averaging
- threshold-based class mapping

For a student project, this is acceptable if you write it like this:
"The final scoring formula was designed for this project by adapting standard concepts from geospatial suitability analysis, ensemble learning, and retail site selection literature."
